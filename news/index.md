# Changelog

## knitexercise 0.4.2

- Setting the parameter `solutions: FALSE` now adds *-questions* at the
  end of the filename before the file extension

## knitexercise 0.4.1

- Bump version of **roxygen2** package used to generate package
  documentation

## knitexercise 0.4.0

- **knitexercise** now requires the minimum version of R to be R 4.0.0
  (because a dependency **knitr**, **evaluate**, now has this
  requirement)

## knitexercise 0.3.0

- Bump version of **roxygen2** used to create the package documentation

## knitexercise 0.2.0

- Bump minimum required version of R to be 3.6.0

## knitexercise 0.1.0

- First version of package
